The Sitemap

Designing the sitemap has probably been the hardest part. Here are the highlights:

  • Index page displaying courses and related assignments
  • Course list page with list of assignments for each course - Teachers have option to add a new course or update a course here
  • Individual course page, w/ list of assignments - Teachers can update the course or add an assignment from here
  • Individual assignment page, where students can upload/test submissions - Teachers can add a test or update the assignment from this page.
